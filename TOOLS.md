
## MemClaw — Fleet Shared Memory

CLI at `/home/goodclaw/.local/bin/memclaw`. Use on every heartbeat.

```bash
memclaw search "query" --limit 5        # search before work
memclaw write "what happened" --type fact  # write after work
memclaw brief "topic"                     # summarized recall
memclaw stats                              # tenant overview
```

- **Tenant:** goodclaw2-at-gmail-com
- **Always include dates in content**
- **Always use your agent_id** via --agent_id flag
